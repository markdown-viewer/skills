## Wireframe Stencils Used

| Component | Stencil / Style |
|-----------|-----------------|
| Image Placeholder | `mxgraph.mockup.misc.image_placeholder` |
| Buttons | Custom rectangles with `rounded=1;fillColor=#007AFF` |
| Cards | `rounded=1;shadow=1;strokeColor=#E5E5E5` |
| Text | Standard text cells with various font sizes |

## Layout Structure

- **Header**: Fixed 70px height, logo + nav links + CTA
- **Hero**: 400px, two-column (text + image)
- **Features**: 3-column card grid
- **CTA Banner**: Full-width colored section
- **Footer**: Multi-column links + copyright
