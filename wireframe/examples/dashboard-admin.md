## Dashboard Layout Pattern

- **Sidebar**: 240px fixed width, dark theme
- **Header**: 70px height with search and profile
- **KPI Cards**: 4-column grid, equal width
- **Charts Row**: 2-column (60%/40% split)
- **Data Table**: Full width
