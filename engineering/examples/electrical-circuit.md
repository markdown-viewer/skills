# Electrical Circuit - Multi-Stage Amplifier

A multi-stage PNP transistor amplifier circuit with RC coupling, inductors, diode protection, and potentiometer bias adjustment.

Based on template: `engineering/electrical_2.drawio`

## Stencils Used

All stencil names from `skills/drawio/stencils/electrical.md`:

| Component | Stencil | Count |
|-----------|---------|-------|
| Resistor | `mxgraph.electrical.resistors.resistor_2` | 6 (R1–R3, R5–R7) |
| Potentiometer | `mxgraph.electrical.resistors.potentiometer_2` | 1 (R4) |
| Capacitor | `mxgraph.electrical.capacitors.capacitor_3` | 6 (C1–C6) |
| PNP Transistor | `mxgraph.electrical.transistors.pnp_transistor_1` | 4 (Q1–Q4) |
| Diode | `mxgraph.electrical.diodes.diode` | 1 (DI) |
| Inductor | `mxgraph.electrical.inductors.inductor_3` | 2 (L1, L2) |
| Ground / Node | `mxgraph.electrical.abstract.dac` | 4 |

- **Component style**: all use `fillColor=#ffffff` (white fill) with `labelBackgroundColor=none`
- **Connections**: orthogonal edges (`edgeStyle=orthogonalEdgeStyle`) with `endArrow=none` (plain wires)

## Circuit Overview

- **4-stage PNP transistor amplifier** (Q1→Q2→Q3→Q4) with RC coupling between stages
- **Input**: inductors L1, L2 form the input coupling/transformer network
- **Bias**: potentiometer R4 provides adjustable bias; resistors R1–R3, R5–R7 set operating points
- **Coupling**: capacitors C1–C6 provide inter-stage AC coupling and bypass filtering
- **Protection**: diode DI provides signal clamping/protection
- **Connections**: orthogonal edges (`edgeStyle=orthogonalEdgeStyle`) with `endArrow=none` (plain wires)
