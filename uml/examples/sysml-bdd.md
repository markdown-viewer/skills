# SysML Block Definition Diagram (BDD)

**Purpose**: Systems Modeling Language (SysML) is an extension of UML for complex system design. Block Definition Diagrams show system structure.

**Key Elements**:
- SysML package: `shape=mxgraph.sysml.package;labelX=150` - labelX controls title width
- Diagram type prefix: `bdd` (Block Definition Diagram), `ibd` (Internal Block Diagram), `req` (Requirements Diagram), `par` (Parametric Diagram)
- Block content: HTML format, use `<hr>` to separate regions (values, parts, constraints, parameters)
- Block labels: `<<block>>`, `<<constraint>>`
- Composition relation: `startArrow=diamondThin;startFill=1;startSize=12` (filled diamond)
- Multiplicity label: Use label element with `connectable="0"`
